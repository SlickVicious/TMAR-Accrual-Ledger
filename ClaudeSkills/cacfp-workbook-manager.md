# CACFP Workbook Manager

**Type:** Claude Desktop Project Skill

CACFP (Child and Adult Care Food Program) meal program tracking workbook manager. Generates monthly templates, adds sites, audits formulas, enters meal data, checks reimbursement calculations. Covers Wayne County sites (Fremont, Goldsboro, Mount Olive) for Boys & Girls Club.

## Trigger phrases
- "CACFP", "meal tracking", "meal counts", "food inventory", "supper counts"
- "next month's sheet", "update the sites", "check the totals"
