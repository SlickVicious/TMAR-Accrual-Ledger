# Financial Document Processing

## Overview

Management system for the LDG vault financial infrastructure, centered on the 13-tab Unified Master Account Register workbook. Covers transaction tracking, tax form management, gap analysis, trust ledger maintenance, and filing chain compliance.

## Master Account Register Architecture

Generator: `Digital File Cabinet/Financials/Master-Account-Register/generate_unified_workbook.py`
Output: `Wimberly_Unified_MasterRegister_2025.xlsx`
Primary platform: Apple Numbers
Backup: Google Sheets at `https://docs.google.com/spreadsheets/d/1H6njThCnOgZ4Obj6e0Iqq-Mkd2fiDAJt/`
GAS Formatting: `06 Toolkit/Scripts/GAS/apply_master_register_formatting.gs`

### 13-Tab Structure

#### Group A — Living Dashboard (7 tabs)

| Tab | Purpose | Gap Scanner Checks |
|-----|---------|-------------------|
| Executive Dashboard | Overview and action items | Flags unresolved action items |
| Transaction Ledger | All financial transactions | Flags empty date/category/amount/status |
| W-2 and Income Detail | Employment/1099 income | Flags Verify or Estimated in Col C |
| BOA Cash Flow | Bank of America flows | Flags empty balance/deposit cells |
| Household Obligations | Bills and recurring expenses | Flags missing status/due/amount |
| Subscriptions and Services | Recurring subscriptions | Flags missing cost/status/tax flag |
| Tax Strategy | Tax planning overview | Flags Verify or Estimated in Col C |

#### Group B — Executive Estate Ledger (6 tabs)

| Tab | Purpose | Gap Scanner Checks |
|-----|---------|-------------------|
| Master Register | All accounts inventory | Flags empty type/status/user |
| Trust Ledger | Trust assets and liabilities | Flags missing value/status |
| 1099 Filing Chain | 1099 filing sequences | Flags incomplete chains |
| Forms and Authority | Regulatory form tracking | Flags Not Started or filed-no-date |
| Proof of Mailing | USPS certified mail tracking | Flags missing tracking/unconfirmed |
| Document Inventory | Physical/digital doc catalog | Flags missing status/tab/location |

## Gap Report System

The Gap Report is generated by GAS script (apply_master_register_formatting.gs, lines 1318-1751). It runs only in Google Sheets and checks cell values, NOT external files.

Run via: TMAR Tools then Data Gap Scanner then Run Full Scan

**Critical**: The scanner reads spreadsheet cells only. It cannot see files on disk, completed tasks outside the sheet, or documents in the vault.

### Common False Positives

| Tab | Cause | Fix |
|-----|-------|-----|
| W-2 and Income Detail | Status still says Verify after docs obtained | Update Col C to Confirmed |
| Executive Dashboard | Action items completed but not removed | Delete completed items |
| Document Inventory | Docs exist on disk but Vault Location empty | Fill Col I with vault path |
| Forms and Authority | Forms filed but date not recorded | Add filing date |
| Proof of Mailing | USPS delivery confirmed but sheet not updated | Update delivery status |

## Tax Form Workflows

### IRS Forms in Vault

Located at `Digital File Cabinet/Gov Forms Links/`:
- EIN application (p1635)
- 1099-A, 1099-B, 1099-C processing
- W-4 correction statements
- Form 2848 (Power of Attorney)
- Form 56/56F (Fiduciary Notice)
- Form 3949-A (Information Referral)
- SSA-7050 (Social Security Earnings)

### 1099 Filing Chain Process

The 1099 Filing Chain tab tracks multi-step filing sequences through stages A, B, and C. The gap scanner flags incomplete chains where one stage is filed but the next has not started.
