## 📈 UNIFIED MASTER REGISTER (2026-02-11)

The original 12-tab design has been superseded by a **Unified 13-tab + _Validation workbook** that merges three data sources and aligns with the Freeway Mechanics (FWM) trust administration framework.

### New Files (generated)
| File | Location |
|------|----------|
| `generate_unified_workbook.py` | `Digital File Cabinet/Financials/Master-Account-Register/` |
| `Wimberly_Unified_MasterRegister_2025.xlsx` | `Digital File Cabinet/Financials/Master-Account-Register/` |
| `apply_master_register_formatting.gs` | `06 Toolkit/Scripts/GAS/` |

### Unified Tab Structure
**Group A — Living Dashboard (7 tabs):** Executive Dashboard, Transaction Ledger, W-2 & Income Detail, BOA Cash Flow, Household Obligations, Subscriptions & Services, Tax Strategy

**Group B — Executive Estate Ledger (6 tabs):** Master Register (28 accounts), Trust Ledger, 1099 Filing Chain, Forms & Authority, Proof of Mailing, Document Inventory

**Hidden:** _Validation (dropdown lists)

### Data Sources Merged
1. Python Financial Workbook (W-2s, BoA cash flow, subscriptions, tax strategy)
2. Wimberly_Master_Transaction_Database.csv (224 transactions)
3. SCALABLE 12-tab design (Master Register columns, dropdowns)
4. FWM course notes (trust categories, filing chain, binder tabs)

### Upload Workflow
1. Run `python3 generate_unified_workbook.py`
2. Upload `.xlsx` to Google Drive -> Open with Sheets
3. Extensions -> Apps Script -> paste `.gs` -> run `applyAllFormatting()`
